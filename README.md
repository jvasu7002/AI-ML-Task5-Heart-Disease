# Decision Trees and Random Forests on Heart Disease Dataset

## Project Overview

This project demonstrates the use of Decision Tree and Random Forest algorithms to predict the presence of heart disease.

---

## Objective

* Build classification models using tree-based algorithms
* Compare Decision Tree and Random Forest performance
* Analyze feature importance

---

## Tools & Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## Dataset

* Heart Disease Dataset
* Target: target (1 = Disease, 0 = No Disease)

---

## Steps Performed

1. Loaded dataset
2. Selected features and target variable
3. Split data into training and testing sets
4. Trained Decision Tree model
5. Controlled overfitting using max_depth
6. Trained Random Forest model
7. Compared accuracy
8. Visualized feature importance
9. Visualized decision tree

---

## Model Performance

* Decision Tree Accuracy: ~0.80–0.85
* Random Forest Accuracy: ~0.85–0.90

Random Forest performed better due to ensemble learning.

---

## Key Learnings

* Working of Decision Trees
* Overfitting and its control
* Ensemble learning using Random Forest
* Feature importance interpretation

---

## How to Run

1. Install dependencies:
   pip install pandas scikit-learn matplotlib

2. Run the script:
   python task5.py

---

## Conclusion

Tree-based models are effective for classification tasks. Random Forest improves performance by combining multiple decision trees.
